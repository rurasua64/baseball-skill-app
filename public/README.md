# Skill FanGraphs Baseball - Detector de Discrepancias F5

Aplicación web para detectar spots de valor en apuestas de MLB (Primeras 5 Entradas) basado en el algoritmo de detección de discrepancias.

## Algoritmo

Basado en el análisis de wRC+ contra abridores específicos y ajuste por calidad de pitchers.
